# B站视频自由缩放

适用于 [哔哩哔哩](https://www.bilibili.com) 视频页的 Tampermonkey 用户脚本。在非全屏状态下，直接拖动播放器边框即可自由调整整块播放区域的画幅比例和页面占位尺寸。

## 功能

- **边框拖拽缩放**：支持拖动播放器上下左右边框，以及四个角落，自由改变整块播放区域的宽高比例。
- **无面板启用**：安装后自动生效，不添加常驻按钮或设置面板。
- **页面布局同步**：横向缩放时同步调整 B 站页面左侧播放区，不只改变内部 video 元素。
- **临时等比缩放**：拖动时按住 `Shift` 可保持当前画幅比例，松开后继续自由缩放。
- **普通模式限定**：网页全屏、浏览器全屏、mini 播放等状态下自动避让。
- **尺寸记忆**：按当前视频记录调整后的尺寸，再次进入同一视频会自动恢复。
- **双击重置**：双击任意拖拽边缘可清除当前视频保存的尺寸，恢复 B 站默认布局。
- **轻量刷新**：拖动过程按动画帧合并更新，停止拖动后再保存尺寸，减少页面抖动。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或兼容的用户脚本管理器。
2. 从 Greasy Fork 安装，或安装开发版：

   ```text
   https://github.com/Wan-JD/bilibili-video-resizer/raw/main/bilibili-video-resizer.user.js
   ```

3. 打开 B 站视频页，鼠标移到播放器边缘，拖动高亮边框即可缩放。

## 使用说明

| 场景 | 操作 |
|------|------|
| 横向放大播放器 | 拖动播放器左边或右边 |
| 调整画幅高度 | 拖动播放器上边或下边 |
| 同时调整宽高 | 拖动播放器四个角 |
| 保持当前比例 | 拖动时按住 `Shift` |
| 恢复默认尺寸 | 双击任意拖拽边缘 |

拖拽时右上角会短暂显示当前宽高。脚本只改当前页面里的播放器和播放区容器样式，不会修改视频源或账号数据。

## 隐私说明

- 不请求任何第三方接口。
- 不上传任何数据。
- 尺寸记录仅保存在 Tampermonkey 本地存储中。

## 开发验证

```text
npm install
npm test
```

测试会在 Chromium 中注入当前用户脚本，覆盖主播放器识别、整块播放区横向缩放、纵向缩放、`Shift` 等比缩放和双击重置。
